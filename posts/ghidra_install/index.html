Install Ghidra on Windows for Dummies

May 26, 2021

I always forget how to install Ghidra whenever I need it on a windows machine, because it’s so simple for linux, I just run the following shell script: https://github.com/bkerler/ghidra_installer

Step 1. Navigate to https://ghidra-sre.org

Step 2. Click “Download Ghidra vX.Y.Z”

Step 3. Right-Click the installed zip file, and click “Extract All…”

Step 4. The folder should be extracted to the download folder, as this works for me and other folders doesn’t. Feel free to play around

Step 5. Navigate to https://java.com

Step 6. Download it

Step 7. Run the executable

Step 8. Click “Install”

Step 9. Search for “Java JDK 11” in a browser

Step 10. Go to the oracle site

Step 11. Click “Download” on the Windows x64 Installer

Step 12. Check off “I reviewed and accept the Oracle Technology Network License Agreement for Oracle Java SE”

Step 13. Click Install

Step 14. If you have a profile, log-in. If you don’t create one and log-in.

[*] Yes this is required

Step 15. Run the executable

Step 16. Now everything should be installed. Try running Ghidra, it should now work. If not, refer to the troubleshotting part of their documentation at: https://ghidra-sre.org/InstallationGuide.html

[*] Should be noted: Don’t change anything, you can’t even change the name of the folder… Smh what even is Windows

Creating the red dragon icon for the shortcut:

Step 1. Create a shortcut and place it on the desktop

Step 2. Rename it so that it doesn’t have a shortcut name

Step 3. Right click on the file and click properties

Step 4. Click “Change Icon”

Step 5. After the error message click “Browse”

Step 6. Navigate to the download folder in your file explorer

Step 7. Search for “*.i”

Step 8. Click the icon, and click open

Step 9. Click okay, apply, and ok again.
