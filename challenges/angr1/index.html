angr1 - baby's first angr (1/5)

September 16, 2021

About challenge

Author: cave

Date created: 2021-09-16

This challenge is an easy challenge I wrote for myself to learn about angr. Provided will be the source code, but no solutions.

Goal: While the flag is obviously {help}, the goal is to get angr to find it, without using any constraints to determine any of the characters. Assume you know only the length, and none of the characters. You want to get the “Its the FLAG” message.

Source code

/*
Author: Cave
Difficulty: 1/5 
*/

#include <stdio.h>
#include <string.h>


int main(){
    char option[5];
    char flag[] = "flag{help}\n";
    char flag_in[50] = {0};

    fgets(option, 5, stdin);
    if (strcmp(option, "lol\n") == 0)
        { 
            printf("[+] First answer is correct! ");
            printf("It's 'lol'!\n");
        
            fgets(flag_in, 50, stdin);

            if (strcmp(flag_in, flag) == 0)
                { 
                    printf("[+] Second answer is correct! ");
                    printf("It's THE FLAGGGG!: '%s'\n", flag);
                }   
        }
        
    return -1; 
}
